# PatternOS Roadmap - Zero-Cost Research Platform

## Vision
Build a **near-zero-cost Indian market research & analysis platform** leveraging free data sources, cached LLM responses, and self-hosted infrastructure for pattern analysis, backtesting, and signal generation.

---

## Guiding Principles

1. **Zero External Cost**: All APIs must be free tier or open-source
2. **Research-First**: Focus on analysis tools, not just alerts
3. **Indian Market**: NSE/BSE stocks + Indian Mutual Funds
4. **Self-Hosted**: PostgreSQL (local), no paid cloud services

---

## Phase 1: Foundation (Zero-Cost Data Pipeline)

### 1.1 Free Data Sources Integration

| Data Type | Source | Cost | Status |
|-----------|--------|------|--------|
| Stock Prices | `yfinance` (Yahoo Finance) | Free | Add to project |
| Stock Prices | `nsetools` / `nsepy` | Free | Add to project |
| MF NAV | AMFI (mfapi.in) | Free | ✅ Existing |
| MF Holdings | mfapi.in | Free | ✅ Existing |
| F&O Data | NSEpy | Free | Add to project |
| Index Data | yfinance | Free | Add to project |

**Implementation:**
- Create unified data fetcher service
- Use caching to avoid repeated API calls
- Fallback chain: yfinance → nsepy → AMFI

### 1.2 Local Data Storage

- Store all fetched data in local PostgreSQL
- 24-hour cache TTL for price data
- Historical data for backtesting (store 2 years)

## Phase 2: Research Tools (Core Features)

### 2.1 Pattern Performance Analytics

**Problem:** Which patterns actually work in Indian market?

**Solution:**
- Track pattern detections over time
- Store outcomes (profit/loss at 1W, 1M, 3M)
- Generate pattern success rate reports
- Compare against benchmarks (Nifty, Sector)

**Zero-Cost:** Uses existing backtesting engine + outcome tracking

### 2.2 Sector Rotation Dashboard

- Track sector performance (NSE sector indices via yfinance)
- Generate sector heatmap (daily)
- Identify rotation signals ( momentum vs laggard)

**Zero-Cost:** yfinance + CSS heatmap visualization

### 2.3 Stock Comparison Tool

- Compare up to 5 stocks side-by-side
- Metrics: P/E, P/B, Debt/Equity, ROE, Dividend Yield, Beta
- Technical: SMA crossover, RSI, MACD status
- Correlation matrix

**Zero-Cost:** yfinance data + local computation

### 2.4 F&O Analysis Module

- Fetch OI data via NSEpy
- Calculate PCR (Put-Call Ratio)
- Identify OI buildup in stocks
- Track F&O specific patterns

**Zero-Cost:** NSEpy (free)

### 2.5 Earnings Calendar

- Track quarterly results dates
- Store estimates vs actuals
- Surprise alerts
- Forward PE calculation

**Zero-Cost:** yfinance + manual entry option

## Specific Implementation Details

### New Dependencies (all free)
```
yfinance>=0.2.0
nsetools>=1.7.0
```

### New API Endpoints
- `GET /api/v1/data/stock/{symbol}` - Price + fundamentals from yfinance
- `GET /api/v1/data/index/{index}` - Index data (Nifty, BankNifty)
- `GET /api/v1/analytics/pattern-performance` - Success rates
- `GET /api/v1/analytics/outcomes` - All tracked outcomes

### Database Extensions
```sql
-- PatternOutcome table
CREATE TABLE pattern_outcomes (
    id SERIAL PRIMARY KEY,
    symbol VARCHAR(20),
    pattern_type VARCHAR(50),
    detected_at TIMESTAMP,
    entry_price DECIMAL,
    holding_period VARCHAR(10), -- 1W, 1M, 3M
    exit_price DECIMAL,
    return_pct DECIMAL,
    outcome VARCHAR(10) -- WIN, LOSS, PENDING
);
```
